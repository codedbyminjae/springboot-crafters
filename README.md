## 📖 About
이 레포지토리는 **Spring Boot 학습 및 실습 코드**를 기록하고,  
마지막에는 하나의 **웹 애플리케이션**을 완성하는 것을 목표로 합니다.  

- 기초부터 차근차근 정리 ✍️  
- 실습 예제 코드 포함 💻  
- 프로젝트 단위로 성장 🚀
